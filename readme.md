# Student Dashboard

A modern, responsive student dashboard web application built with **Flask**, **Bootstrap 5**, and **JavaScript**.  
Designed to manage courses, track grades, and visualize performance analytics.

---

## Features

- **Dashboard Overview**: Quick access to courses, grades, and analytics.
- **Add Course**: Easily add new courses to your profile.
- **Analytics**: View charts and performance stats.
- **GPA Calculator**: Calculate your GPA based on your course grades.
- **Grade Predictor**: Estimate future grades.
- **Dark/Light Mode**: Toggle between themes.
- **Quotes Widget**: Motivational quotes rotate automatically.
- **Responsive Design**: Works on desktop and mobile devices.
